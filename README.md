# ML_WORK1
Pedro Rici - 20241028465


Repositório para o primeiro trabalho da disciplina da Aprendizado de Máquina do Programa de Pós Graduação em Engenharia ELétrica e de Computação da Univerisdade Federal do Rio Grande do Norte para o semestre 2024.2 que consiste em 5 vídeos de até 5 minutos resumindo os 5 capítulos do livro "Desinging Machine Learning Systems" de Chip Huyen.

Para esse trabalho, a metodologia utiliza foi ler os capítulos propostos (1 ao 5) e resumir cada capítulo destacando seus principais pontos.
Após isso, utilizando o Canva foi criado um slide de base para o vídeo, que foi feito com base nos resumos criados e em imagens do próprio livro.
Após isso foi feito a gravação, o Canva permite gravar página por página do slide, de forma que o controle do tempo máximo e a alteração por partes se torna mais simplicada e rápida.
